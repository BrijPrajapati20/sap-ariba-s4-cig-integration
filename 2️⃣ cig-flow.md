# 🔄 CIG Integration Flow

## 📌 Objective
Understand how documents move via CIG.

---

## 🔁 Flow

### Ariba → S4
- PR/PO created  
- Sent to CIG  
- Mapping applied  
- Delivered to S4  

### S4 → Ariba
- Invoice/Confirmation  
- Sent via CIG  
- Updated in Ariba  

---

## ⚙️ Key Activities

- Data mapping  
- Message transformation  
- Routing  

---

## ⚠️ Failure Points

- Mapping errors  
- Connectivity issues  
- Data inconsistency  

---

## 🎯 Learning

CIG is the backbone of integration — understand flow deeply.
