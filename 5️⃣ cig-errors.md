# ❌ Common CIG Errors

## 🔍 Types of Errors

### 1. Connectivity Error
- Endpoint not reachable  
- Network issue  

### 2. Mapping Error
- Data mismatch  
- Missing fields  

### 3. Authentication Error
- Certificate expired  
- Credential issue  

---

## ✅ Resolution Approach

1. Check logs in CIG  
2. Identify error type  
3. Validate configuration  
4. Fix and reprocess  

---

## 🎯 Learning

Always categorize error before fixing.
