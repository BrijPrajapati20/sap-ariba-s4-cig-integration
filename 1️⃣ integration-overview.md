# 🏗️ Integration Overview

## 📌 Objective
Enable seamless communication between SAP Ariba and S4 system.

---

## 🔗 Components

- SAP Ariba (Frontend Procurement)  
- CIG (Middleware)  
- SAP S4 (Backend System)  

---

## 🔁 High-Level Flow

1. Document created in Ariba  
2. Sent to CIG  
3. Transformed & routed  
4. Received in S4  

---

## 🎯 Key Points

- CIG acts as middleware  
- Handles transformation & routing  
- Ensures secure communication  

---

## 🧠 Learning

Understanding architecture is key before debugging issues.
